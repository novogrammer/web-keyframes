# AGENTS

## 目的

このリポジトリは、Web アニメーション向けの軽量なキーフレームデータエディタ兼 SCSS 生成ツール `web-keyframes` を扱います。

## 作業ルール

- このプロジェクトのスコープは狭く保つこと。ユーザーが明示的に求めない限り、汎用的なタイムラインツールやシーケンサーへ拡張しない。
- 広い UI 抽象化より、明示的で小さなデータモデル変更を優先する。
- transform の挙動は、内部データモデル、SCSS 生成、preview 生成、エディタ UI のあいだで一致させる。
- キーフレームのデータ形式を進化させる場合も、可能な限り timeline JSON の後方互換を維持する。

## ドキュメント更新ルール

- ドキュメント更新は冪等に行うこと。
- 同じ指示を複数回実行しても、同じ内容が重複追記されないようにすること。
- 新しい情報は既存の該当箇所へ統合し、別の場所へ重複して追記しないこと。
- 古い記述、重複、矛盾は、ドキュメント更新時に削除または更新すること。
- タスク完了時は、完了項目を `docs/todo.md` に残したままにせず、`docs/done.md` へ移すこと。

## 検証

- コード変更時は、ドキュメント専用の変更でない限り `npm test` による確認を優先する。
- 公開データ形式やエディタ操作に影響する変更を行った場合は、関連する README の例とテストも同じ変更内で更新する。
